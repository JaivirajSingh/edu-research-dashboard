# A Comparative Analysis of Urban and Rural Government Schools in Jammu: Interactive Dashboard

A static, narrative-driven data visualization dashboard analyzing the educational divide between urban and rural government schools in Jammu. 

## Research Foundation
This dashboard is the technical implementation of the Major Project Report: **"A Comparative Analysis of Urban and Rural Government Schools in Jammu: Perspectives from NEP 2020 and SDG 4."** Conducted under the Four-Year Undergraduate Programme (Design Your Degree) at the Skill Incubation Innovation Entrepreneurship Development Centre (SIIEDC), University of Jammu.

## Technical Stack
- **Framework:** React.js (via Vite)
- **Visualisation:** Chart.js 
- **Styling:** CSS

## Data Management & Architecture
**Note:** This is a strictly static dashboard. 
There is no external database, API integration, or dynamic data fetching. All narrative text, metrics, and chart configurations are hardcoded within `src/storyData.js`. The architecture was intentionally constrained to prioritize immediate frontend presentation over backend data management.
